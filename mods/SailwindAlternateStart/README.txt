Requires Mod Manager and requires the SailwindModConsole Mod

Console Commands in this mod-

"start goldrock"  	-start at goldrock
"start dragon"     	-start at dragon cliffs
"start fort"       	-start at fort aestrin

This mods adds console commands which allow you to start the game in a capital region with enough money to buy a large ship.
Simply type in the command and exit the pause menu to begin.
