This mod adds a mod console to Sailwind for other mods to use, it uses Sailwind Mod Manager.
It is not fully populated with commands yet but also allows other mods to use it to send commands.

Installation: copy the SailwindModConsole folder into your Mods folder

Uninstallation: delete the SailwindModConsole folder from your Mods folder

Commands(For now):

"setgold_X" 	sets your gold to ammount specified
"setthirst_X"	sets thirst bar to value specified(1-100)
"sethunger_X"	sets hunger bar to value specified(1-100)
"setsleep_X"	sets sleep bar to value specified(1-100)
"getweight"		displays mass of currently held item
"timeofday" 	displays current local & world time
